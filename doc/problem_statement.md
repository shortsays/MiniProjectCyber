## Title
Network-Based Phishing Detection Tool

## Abstract
Phishing attacks are a significant cybersecurity threat where attackers use fake websites to steal sensitive information. This project proposes a network-based phishing detection system that captures HTTP and DNS traffic and analyzes domain patterns to identify phishing attempts in real time.

## Tools Used
- Linux
- Python
- Scapy
- Requests

## Outcome
The system detects suspicious domains and alerts users, helping prevent credential theft.
